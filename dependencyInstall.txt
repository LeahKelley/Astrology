Windows Install Python Virtual Environment (WSL):
sudo apt install python3.12 python3.12-venv python3.12-dev build-essential -y
python3.12 -m venv .venv (This should be done already)

Windows (WSL):
source .venv/bin/activate
pip freeze > requirements.txt
pip install -r requirements.txt
deactivate //to get out of virtual environment
